# Vite js admin template (CSR)

- Primevue 4
- Tailwind
- Vueuse
- Pinia
- Eslint 8 + Prettier
- Vue Router 4

### To Dev

1. `yarn install`
2. `yarn dev`

### To Build

1. `yarn build`

### To format or linting

1. `yarn lint` or `yarn format`

### Dev Environment

- Node v18.20.4
- Npm 10.7.0
- Yarn 1.22.22
